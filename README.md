# book-search-engine

## Description:
The Book search engine uses the MERN stack with a React front end, MongoDB database, Node js and Express js server. It is built using Google Books API and enables users to save book searches to their profile for future reference. Also, the search engine is built with GraphQL API and Apollo Server.

  ## Installation:

   Make sure you have MongoDB installed on your machine. First clone the repo, install dependencies with npm i and then run ```npm start``` to run the server.
